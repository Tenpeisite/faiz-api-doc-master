## SpringBoot

### 🍊Maven
在项目的pom.xml的dependencies中加入以下内容:
```xml
<dependency>
  <groupId>com.zhj</groupId>
  <artifactId>API-client-sdk</artifactId>
  <version>0.0.1</version>
</dependency>
```
### 🍐Gradle
```
implementation group: 'com.zhj', name: API-client-sdk', version: '0.0.1'
```
